# Privacy Policy

## Data Processing

This MCP server is a **read-only regulatory reference tool**. It:

- Processes no personal data
- Stores no user queries or session data
- Contains no analytics, telemetry, or tracking
- Makes no outbound network requests during normal operation (all data is served from a local SQLite database)
- Requires no user accounts or authentication

## Data Stored

The server's SQLite database contains only:

- BIS IS standards catalog metadata (IS numbers, titles, TC codes, ICS codes, adoption years, ISO equivalents) — publicly available
- IT Act rules and publicly available regulatory framework summaries

No BIS standards full text is stored (that is paid/licensed content). No personal data is stored.
